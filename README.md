# BRACU CSE330: Numerical Methods Laboratory

A comprehensive repository containing lab implementations and solutions for the CSE330 Numerical Methods course at BRAC University. This collection showcases the numerical analysis techniques and algorithms learned throughout the course, implemented as part of the laboratory assignments.

## Course Overview

This repository contains practical implementation of fundamental numerical methods and algorithms covered in the CSE330 Numerical Methods course at BRAC University. The implementations focus on computational techniques for solving mathematical problems that cannot be solved analytically, including interpolation, differentiation, integration, solving nonlinear equations, and numerical linear algebra.

## Chapter 2: Polynomial Interpolation
# CAUCHY‚ÄôS THEOREM WITH NO INTERVAL
Practice Problems: https://drive.google.com/file/d/1UDJs40cc9BJpVxaGxhi9_4R1wx6NPNFq/view?usp=drive_link 

Suppose you are finding the max truncation error with Cauchy‚Äôs Thorem and you want to maximise the value of $\frac{f^{(n+1)}(\xi)}{n!}$, however, you come up accross a term
f^{(n+1)}(\xi) = 2cos(\xi)+sin(\xi) \dots(1)
So how would you maximise it?

- If the interval [a,b] is provided then you go thorough the regular process of checking the value of the $f^{(n+1)}(\xi)$ at each interval. So, basically the maxmim value is $max(f^{(n+1)}(a), f^{(n+1)}(b))$
- If not interval provided, you maximise each component separately, as no value can be larger than that the sum of each of the terms maximum value. So, In the equation(1) we have two terms.  $2cos(\xi)\  \& \  sin(\xi)$. So, the maximum value of $max(2cos(\xi)) = 2 \times\cos(0)= 2\times 1 = 2$. And, $max(\sin(\pi/2)) = 1$. So the maximum of
max(f^{(n+1)}(\xi) ) = 2 \times 1+1 = 3
